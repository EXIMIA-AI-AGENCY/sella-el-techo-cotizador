# Sella El Techo - Cotizador

Cotizador web para servicios de sellado de techos.

## Precios

- **Precio base:** $3.50 por pie cuadrado

### Servicios Adicionales (Add-ons)

| Servicio | Precio |
|----------|--------|
| Remover Aire Acondicionado | $350.00 |
| Remover Placas Solares | $500.00 |
| Remover Cisterna | $250.00 |
| Remover Antena/Satélite | $150.00 |
| Limpieza Profunda del Techo | $200.00 |

## Uso

Abrir `index.html` en un navegador web.

## Archivos

- `index.html` - Estructura HTML del cotizador
- `styles.css` - Estilos visuales
- `calculator.js` - Lógica de cálculo
